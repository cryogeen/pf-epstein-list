# pf-epstein-list
A big txt file including all discord user ids of people in this community who should be banned.


# Removal

To request a removal, open an issue.

You must provide a solid argument as to why this person should be removed.\
If no valid argument is to be had, the user id shall stay up.

# Addition

To request an addition, open an issue.

I do not accept random additions with no proof.
